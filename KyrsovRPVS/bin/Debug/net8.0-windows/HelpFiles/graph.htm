Интерпретация графика решения

После вычисления решения система строит в PictureBox график значений переменных xi:

  • Ось X (горизонтальная) показывает номера переменных: x1, x2, …, xn.
  • Ось Y (вертикальная) показывает численные значения этих переменных.

Что вы видите на графике:

  1. Точками обозначены вычисленные значения xi.
  2. Линиями (если включена) соединены соседние точки для наглядности тренда.
  3. Под каждой точкой внизу подпись с номером переменной (x1, x2,…).

Пример

Для решения n = 3 с результатами x = [1, 2.5, –0.7] вы получите:

  • Точка (1; 1)
  • Точка (2; 2.5)
  • Точка (3; –0.7)

Этот график помогает наглядно сопоставить полученные значения переменных.
