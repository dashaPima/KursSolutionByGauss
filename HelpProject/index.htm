Системы линейных алгебраических уравнений и метод Гаусса

Что такое система линейных алгебраических уравнений

Система линейных алгебраических уравнений (СЛАУ) — это набор уравнений вида: 

        a??x? + a??x? + ... + a?nxn = b?  
        a??x? + a??x? + ... + a?nxn = b?  
        ...  
        am?x? + am?x? + ... + amnxn = bm

Здесь a?? — коэффициенты при переменных x?, x?, ..., xn, а b? — свободные члены. Задача — найти такие значения переменных x?, x?, ..., xn, при которых все уравнения выполняются одновременно.

Описание метода Гаусса

Метод Гаусса — это пошаговая процедура приведения системы уравнений к так называемому треугольному виду, после чего производится обратный ход для нахождения значений переменных. Метод основан на элементарных преобразованиях строк расширенной матрицы:

  1. Перестановка строк местами
  2. Умножение строки на ненулевое число
  3. Прибавление к строке другой строки, умноженной на число

Этапы метода

1. Прямой ход (приведение к верхнетреугольному виду)

Сначала с помощью элементарных преобразований устраняются переменные из всех уравнений ниже первой строки. Повторяя процесс по диагонали, в результате получаем матрицу, в которой ниже главной диагонали — нули.

2. Обратный ход (выражение переменных)

Начиная с последнего уравнения (в котором только одна переменная), выражаем переменные "снизу вверх", подставляя уже найденные значения.

Преимущества метода

  • Универсальность — подходит для любой невырожденной квадратной системы
  • Простота алгоритма — легко реализуется программно
  • Базовый метод, на котором строятся более продвинутые алгоритмы (LU-разложение, методы с частичным выбором и т.д.)

Ограничения

При плохой обусловленности матрицы возможны значительные численные ошибки. В таких случаях рекомендуется использовать модификации с частичным или полным выбором главного элемента.

Вывод: Метод Гаусса — это фундаментальный и широко используемый алгоритм решения СЛАУ, особенно эффективный при работе с небольшими и средними по размеру системами.
