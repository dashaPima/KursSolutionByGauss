Ввод системы линейных уравнений

Для решения системы методом Гаусса необходимо задать коэффициенты матрицы A и вектор свободных членов B. Это делается на форме в таблице (DataGridView), обозначенной как «Ввод уравнения».

Порядок действий:

  1. В поле «Установка числа переменных» введите целое число n — размерность системы.
  2. Нажмите кнопку «Построить таблицу». Появится таблица dgvSystem с n столбцами для коэффициентов и одним столбцом для свободных членов.
  3. Столбцы таблицы имеют заголовки:
    • A1, A2, …, An — коэффициенты при переменных x1, x2, …, xn
    • B — свободные члены
  4. Заполните каждую строку таблицы значениями коэффициентов и свободного члена.
  5. После ввода всех данных нажмите кнопку «Найти решение».

Пример заполнения для n = 3:

A1 A2 A3 B
2 1 -1 8
-3 4 2 -2
1 -2 5 3

Контекстная помощь

Если курсор находится в таблице и вы нажмёте F1, появится эта тема справки с указаниями по вводу данных.
